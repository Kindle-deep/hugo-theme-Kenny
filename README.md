## 下载与使用
### 下载
```bash
git init
it submodule add --depth 1 https://github.com/Kindle-deep/hugo-theme-Kenny.git themes/Kenny
```

### 使用
1. 替换config.yaml
```bash
rm config.toml && cp themes/Kenny/exampleSite/config.toml config.toml
```

2. 运行
```bash
hugo server -D
```
### 升级
```bash
git submodule update --rebase --remote
```
